A simple **Mollie.nl** plugin for a direct payment. Interaction with **WP Availability Calendar & Booking** plugin 

== Description ==
The short code [mollie] adds a payment form to the page.

Obtain the API Key form your Mollie.nl account. For testing Mollie supply a test API Key.

The Description option allows you to have a free format description or a Checkin Date with the format yyyy-mm-dd. The Checkin Date is required for interaction with the **WP Availability Calendar & Booking** plugin

== Installation ==

1. Upload the entire jm_mollie_client folder to the /wp-content/plugins/directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the settings in Admin->settings->WP Availability & Bookings Settings
